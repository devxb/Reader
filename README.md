<h3>좋은 개발자란 무엇일까?</h3> 
<p> <i>유지보수에 용이한 코드를 작성하는 사람? 협업을 잘하는사람?</i>
<br> 좋은 개발자에대한 생각은 모두가 다르며, 많은 문장으로 정의될수있다.
<br>이 같은 추상적인 목표에 대해 서로가 읽은 책을 공유하며 함께 성장해 나가는것을 목표로한다.
</p>
<h4> "좋은 개발자"라는 목표에 모인 뉴비들이 만들어 나가는 책 기록 저장소이다.</h4>
